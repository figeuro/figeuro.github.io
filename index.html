Figeuró

Special Characters Without the Hassle.

If you're a student of modern foreign languages, then you definitely know the struggles of inserting special characters into your text.
How much effort are you really willing to put in to achieve this seemingly simple task? Isn't there a better way?

With Figeuró, you can type in French, German and Spanish using just your English keyboard.

Download for Windows
Figeuró is designed to work anywhere that you can enter text. That includes browsers, websites, Microsoft Office products, everywhere.
You'll never forget the combinations you need to type, because they're all very logical - accents use slashes in the same direction (/ is acute, \ is grave), the Umlaut uses the colon and similarly, all characters are typed with keys that visually represent them, which makes sense! Nevertheless, here's a list:
Character Key
Acute Accent #/ with VOWEL
Grave Accent #\ with A, E, O or U
Circumflex #^ with VOWEL
Umlaut (Diaeresis) #: with A, O or U
Combined Letters (French) #+ with AE or OE
C Cedilla #,C
Esszett (ß) #+ss
N with Tilde #~N
French Quotes #<< or #>>
German Lower Quotes #:"
Inverted Punctuation (Spanish) #~! or #~?
Technical Information +
  • If you want to type a key without it transforming into its associated character (ie. you want to type #/e without creating é, click away from the text entry after the hash tag.
  • The program uses ALT codes to insert characters - this is a windows program only!
  • To make sure the program runs at startup, please make a link in your startup folder to the program executable.
  • Double click the executable to start. You won't see anything appear but you'll see the Figeuró icon in your windows notification area icons (near the time display)
